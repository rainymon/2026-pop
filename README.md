# 전국 인구구조 유사 지역 찾기

주민등록 인구 데이터를 활용해 사용자가 선택한 지역의 연령별 인구구조를 시각화하고, 전국에서 인구구조가 가장 비슷한 지역 Top 5를 찾아주는 Streamlit 웹앱입니다.

시도, 시군구뿐만 아니라 읍·면·동 단위 지역도 검색하고 선택할 수 있습니다.

---

## 주요 기능

* 지역명 검색 및 선택
* 시도, 시군구, 읍·면·동 단위 분석
* 선택 지역의 0세부터 100세 이상까지 연령별 인구구조 확인
* Plotly 기반의 인터랙티브 꺾은선 그래프
* 전국 행정구역과 인구구조 유사도 계산
* 인구구조가 가장 비슷한 지역 Top 5 제공
* Top 5 유사도 막대그래프
* 선택 지역과 유사 지역의 연령별 인구구조 비교
* 인구 구성비와 실제 인구수 그래프 전환
* 읍·면·동만 비교하는 기능
* 비교 대상 지역의 최소 인구 설정
* 분석 결과 CSV 다운로드
* CP949, EUC-KR, UTF-8 인코딩 자동 처리
* Streamlit Cloud 배포 지원

## 사용 데이터

이 프로젝트는 행정안전부 주민등록 인구통계 형식의 CSV 데이터를 사용합니다.

기본 데이터 파일명은 다음과 같습니다.

```text
202606_202606.csv
```

데이터에는 다음과 같은 열이 포함되어 있어야 합니다.

```text
행정구역
2026년06월_계_총인구수
2026년06월_계_0세
2026년06월_계_1세
...
2026년06월_계_99세
2026년06월_계_100세 이상
```

열 이름 앞의 연월은 달라질 수 있습니다. 앱은 열 이름의 뒷부분을 기준으로 필요한 열을 자동으로 찾습니다.

예를 들어 다음 열을 모두 인식할 수 있습니다.

```text
2026년06월_계_0세
2026년07월_계_0세
2027년01월_계_0세
```

## 사용 방법

### 1. 데이터 불러오기

앱은 기본적으로 `main.py`와 같은 폴더에 있는 다음 파일을 사용합니다.

```text
202606_202606.csv
```

사이드바의 파일 업로드 기능을 이용하면 다른 주민등록 인구 CSV 파일도 사용할 수 있습니다.

### 2. 지역 검색

검색창에 지역명의 일부를 입력합니다.

예시:

```text
서울특별시
종로구
수원시
청운효자동
정자동
```

검색 결과에서 원하는 지역을 선택합니다.

### 3. 비교 조건 설정

사이드바에서 비교 조건을 설정할 수 있습니다.

#### 모든 행정구역

시도, 시군구, 읍·면·동을 모두 비교합니다.

#### 읍면동만 비교

읍·면·동 단위 지역만 비교합니다.

#### 비교 지역 최소 인구

인구가 너무 적은 지역을 비교 대상에서 제외합니다.

기본값은 다음과 같습니다.

```text
1,000명
```

인구가 매우 적은 지역은 특정 연령 인구가 0명 또는 1명인 경우가 많아 인구구조 유사도가 불안정하게 계산될 수 있습니다.

### 4. 결과 확인

앱에서는 다음 결과를 확인할 수 있습니다.

* 선택 지역의 총인구
* 0~14세 인구 비율
* 15~64세 인구 비율
* 65세 이상 인구 비율
* 인구가 가장 많은 연령
* 인구구조가 비슷한 지역 Top 5
* 각 지역의 유사도 점수
* 선택 지역과 Top 5 지역의 연령별 그래프

### 5. 결과 다운로드

분석 결과를 CSV 파일로 다운로드할 수 있습니다.

다운로드 파일에는 다음 항목이 포함됩니다.

* 기준 연월
* 기준 지역
* 기준 지역 총인구
* 유사 지역 순위
* 유사 지역명
* 유사 지역 총인구
* 인구구조 유사도 점수

## 인구구조 유사도 계산 방법

지역마다 전체 인구 규모가 다르기 때문에 실제 인구수만으로 비교하면 인구가 많은 지역끼리 비슷하게 나타날 수 있습니다.

이 프로젝트는 지역 규모가 아니라 연령별 인구구조의 형태를 비교하기 위해 연령별 구성비를 사용합니다.

### 1. 연령별 인구 벡터 생성

각 지역의 연령별 인구를 다음과 같은 벡터로 만듭니다.

```text
[0세 인구, 1세 인구, 2세 인구, ..., 99세 인구, 100세 이상 인구]
```

총 101개의 연령 구간을 사용합니다.

### 2. 연령별 구성비 계산

각 연령 인구를 해당 지역의 전체 연령 인구 합계로 나눕니다.

```text
연령별 구성비 = 해당 연령 인구 ÷ 지역 전체 인구
```

예를 들어 어떤 지역의 총인구가 10,000명이고 20세 인구가 150명이라면 다음과 같이 계산합니다.

```text
20세 구성비 = 150 ÷ 10,000 = 0.015
```

백분율로 표현하면 1.5%입니다.

### 3. 코사인 유사도 계산

선택 지역과 다른 지역의 연령별 구성비 벡터 사이에 코사인 유사도를 계산합니다.

코사인 유사도는 두 벡터의 방향이 얼마나 비슷한지를 나타냅니다.

```text
코사인 유사도 =
두 벡터의 내적
÷
두 벡터 크기의 곱
```

점수가 1에 가까울수록 두 지역의 연령별 인구구조가 비슷합니다.

앱에서는 보기 쉽게 100을 곱해 백분율 형태로 표시합니다.

```text
유사도 0.9875 → 98.75%
```

### 유사도 해석

|    유사도 | 해석                   |
| -----: | -------------------- |
| 99% 이상 | 연령별 인구구조가 매우 유사함     |
| 97~99% | 상당히 유사함              |
| 95~97% | 전반적인 구조가 유사함         |
| 95% 미만 | 일부 연령대에서 차이가 있을 수 있음 |

이 기준은 절대적인 통계 기준이 아니라 결과를 이해하기 위한 참고용입니다.

## 그래프 설명

### 유사 지역 Top 5 막대그래프

선택 지역과 가장 비슷한 5개 지역을 유사도 순서대로 보여줍니다.

그래프에 마우스를 올리면 다음 정보를 확인할 수 있습니다.

* 지역명
* 유사도 점수
* 총인구

### 연령별 인구 구성비 꺾은선 그래프

선택 지역과 Top 5 지역의 연령별 인구 구성비를 함께 보여줍니다.

지역별 총인구가 달라도 인구구조의 형태를 직접 비교할 수 있습니다.

### 실제 인구수 꺾은선 그래프

연령별 실제 인구수를 보여줍니다.

지역 간 인구 규모 차이를 함께 확인할 때 유용합니다.

유사도 자체는 그래프 표시 방식과 관계없이 연령별 인구 구성비를 기준으로 계산됩니다.

## 데이터 파일 변경 방법

새로운 월의 인구 데이터를 사용할 때는 두 가지 방법이 있습니다.

### 방법 1. 기존 파일 교체

새 CSV 파일의 이름을 다음과 같이 변경합니다.

```text
202606_202606.csv
```

기존 파일을 새 파일로 교체하면 코드 수정 없이 사용할 수 있습니다.

### 방법 2. 코드의 기본 파일명 수정

`main.py` 상단에서 다음 값을 찾습니다.

```python
DEFAULT_CSV_FILE = "202606_202606.csv"
```

새 파일명으로 수정합니다.

```python
DEFAULT_CSV_FILE = "202607_202607.csv"
```

GitHub에도 같은 이름의 CSV 파일을 업로드해야 합니다.

## 인코딩 오류 해결

행정안전부 CSV 파일은 일반적으로 CP949 인코딩을 사용하는 경우가 많습니다.

이 앱은 다음 인코딩을 순서대로 자동 확인합니다.

```text
CP949
EUC-KR
UTF-8-SIG
UTF-8
```

파일을 읽지 못한다면 CSV 파일이 손상되지 않았는지 확인해야 합니다.

엑셀에서 다시 저장할 경우 `CSV UTF-8` 형식으로 저장하면 호환성이 좋아질 수 있습니다.

## 주의사항

### 행정구역 수준 혼합

`모든 행정구역`을 선택하면 시도, 시군구, 읍·면·동이 함께 비교됩니다.

예를 들어 하나의 읍·면·동이 특정 시군구나 광역자치단체와 높은 유사도를 보일 수도 있습니다.

같은 행정구역 수준끼리 비교하고 싶다면 `읍면동만 비교`를 선택하는 것이 좋습니다.

### 인구가 적은 지역

인구가 매우 적은 지역은 한두 명의 차이만으로 연령별 구성비가 크게 달라질 수 있습니다.

비교 지역 최소 인구를 적절히 설정하는 것이 좋습니다.

권장 범위는 다음과 같습니다.

```text
읍면동 비교: 1,000명 이상
시군구 비교: 10,000명 이상
```

### 총인구가 비슷하다는 의미는 아님

유사도 점수는 연령별 인구구조의 모양을 비교한 값입니다.

유사도 점수가 높더라도 두 지역의 실제 총인구는 크게 다를 수 있습니다.

### 데이터 기준 시점

모든 비교는 사용한 CSV 파일의 기준 연월을 따릅니다.

서로 다른 기준 월의 데이터를 하나의 파일에 섞어 사용하면 정확한 비교가 어려울 수 있습니다.

## 실행 오류 해결

### `ModuleNotFoundError`

필요한 라이브러리가 설치되지 않은 경우 발생합니다.

```bash
pip install -r requirements.txt
```

Streamlit Cloud에서는 `requirements.txt`가 저장소에 있는지 확인합니다.

### CSV 파일을 찾을 수 없음

다음 파일이 `main.py`와 같은 위치에 있는지 확인합니다.

```text
202606_202606.csv
```

파일명이 다르면 `DEFAULT_CSV_FILE` 값을 수정해야 합니다.

### 데이터 열을 찾지 못함

CSV 파일에 다음과 같은 연령별 전체 인구 열이 존재하는지 확인합니다.

```text
_계_0세
_계_1세
...
_계_100세 이상
```

열 이름 앞의 연월은 달라도 되지만 `_계_나이` 형식은 유지되어야 합니다.

### Streamlit Cloud에서 앱이 멈춤

CSV 파일 크기가 크거나 분석 대상 지역이 많은 경우 최초 실행 시 데이터 처리에 시간이 필요할 수 있습니다.

앱에서는 `st.cache_data`를 사용하므로 같은 파일을 다시 분석할 때는 처리 속도가 빨라집니다.

## 프로젝트 요약

이 프로젝트는 주민등록 인구 데이터를 활용해 선택 지역의 연령별 인구구조를 분석하고, 전국에서 가장 비슷한 인구구조를 가진 지역을 찾아주는 웹앱입니다.

단순히 인구수가 비슷한 지역을 찾는 것이 아니라 각 연령이 차지하는 비율을 비교하기 때문에, 지역의 세대 구성과 고령화 정도가 비슷한 곳을 탐색하는 데 활용할 수 있습니다.
